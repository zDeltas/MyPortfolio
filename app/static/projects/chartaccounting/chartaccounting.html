Pour mon stage de 2ème année, ma mission durant le stage a été de mettre à disposition une interface web pour afficher des statistiques sous forme graphique. En poursuivant mes études dans le domaine du développement.

Les scientifiques ont besoin de statistiques sur leur utilisation du centre de calcul. Lorsqu'un chercheur a besoin d'effectuer un calcul, il soumet un job au gestionnaire de jobs (ou système de batch, ici GridEngine).

Le projet est séparé en 2 parties :

  • le back office assure la création de la base de données, la maintenance. C'est également ici que l'on parcourt le fichier comptable pour insérer les données dans la base de données.
  • Le front office sera la partie web. Ici, les utilisateurs pourront consulter leurs statistiques de calcul. Pour la recherche, un choix de filtres sera mis à leur disposition.

Ce stage m'a permis de découvrir la programmation avec le module Flask en python.
